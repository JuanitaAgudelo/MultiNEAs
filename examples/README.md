# MultiNEAs Examples

This directory contains example scripts and notebooks demonstrating the usage of MultiNEAs.

## Contents

Example files will be added as the package develops. They will include:

- Jupyter notebooks with tutorials
- Python scripts demonstrating key features
- Data analysis examples

## Running Examples

To run the examples, make sure you have MultiNEAs installed:

```bash
pip install multineas
```

Or install from source in development mode:

```bash
cd MultiNEAs
pip install -e .
```

Then navigate to this directory and run the examples:

```bash
cd examples
python example_script.py
```

For Jupyter notebooks:

```bash
jupyter notebook
```
